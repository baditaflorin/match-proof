# Contributing

Thanks for helping improve Match Proof.

## Local Setup

1. Install Node.js 22+.
2. Run `npm install`.
3. Run `make install-hooks`.
4. Use `make dev` for local development.
5. Run `make test smoke` before pushing.

## Commits

Use Conventional Commits:

- `feat: add matching flow`
- `fix: handle invalid offer`
- `docs: update privacy notes`
- `test: cover bloom filters`

Do not commit secrets, private keys, `.env` files, or local credentials.
